# Task4 最小命令模板（复制用）

本目录脚本**不是**官方主线入口；主线为 `python code/d4c.py …` 与 `scripts/entrypoints/*.sh`。  
将模板复制到本地后按需改变量再执行。

| 文件 | 作用 |
|------|------|
| `task4_step3_min.sh` | Step3 |
| `task4_step4_min.sh` | Step4（须 `--eval-profile`） |
| `task4_step5_min.sh` | Step5（须 `--eval-profile`；`auto` 时须 `STEP4_RUN`） |
| `task4_eval_min.sh` | Eval |
| `task4_eval_matrix_min.sh` | `eval-matrix` |
| `task4_eval_rerank_matrix_min.sh` | `eval-rerank-matrix` |

正式产物根仅为 `runs/task{T}/vN/...`。
